# نظام إدارة المخزون

## التثبيت والتشغيل

1. فك ضغط الملف
2. تثبيت التبعيات:
   ```bash
   npm install
   # أو
   bun install
   ```

3. تشغيل التطبيق:
   ```bash
   npm run dev
   # أو
   bun dev
   ```

## الميزات
- نظام مصادقة بـ 3 أدوار (RFS, OC, RM)
- إدارة مخزون لـ 90+ منتج
- نظام طباعة ملصقات (دعم TM-T88V)
- 8 تبويبات رئيسية
- دعم 3 لغات (العربية، الإنجليزية، الفرنسية)
- نظام تنبيهات تواريخ الانتهاء
